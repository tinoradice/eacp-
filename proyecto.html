Así se ve la experiencia de uso

La aplicación está diseñada para mostrar el estado del sistema, alertar al usuario y recomendar una acción inmediata frente a la detección de gas o monóxido.

Pantallas principales

Esta sección muestra cómo sería la interacción del usuario con SentinelGas desde el celular, con una interfaz clara y pensada para actuar rápido.

Pantalla principal

Muestra el estado general del sistema, el nivel detectado y los accesos principales al monitoreo y las alertas.

Modo alerta

Cuando se detecta una situación de riesgo, la app cambia de estado y muestra indicaciones claras para ventilar, evacuar o tomar una decisión segura.

¿Qué busca demostrar la app?

Más que una interfaz visual, la aplicación representa la conexión entre el sistema técnico y el usuario final, transformando datos en decisiones comprensibles.

📲

Monitoreo

Permite observar el estado del sistema y conocer si existe una situación normal o de riesgo.

🔔

Alertas

Muestra advertencias inmediatas cuando el sistema detecta niveles peligrosos.

🧭

Guía

Propone acciones claras para que el usuario sepa cómo responder en cada caso.

⚡

Velocidad

Está pensada para reducir el tiempo de reacción frente a una emergencia.
