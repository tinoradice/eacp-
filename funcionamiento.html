Funcionamiento del sistema

Detección, análisis, alerta y respuesta

Cómo trabaja SentinelGas paso a paso

SentinelGas convierte datos del entorno en información útil para prevenir accidentes, alertar rápidamente y asistir al usuario con una respuesta clara.

Arquitectura del sistema

1. Detección

Los sensores analizan continuamente el ambiente.

2. Procesamiento

El microcontrolador interpreta los datos recibidos.

3. Alerta

Se activa alarma y notificación al usuario.

4. Acción

El sistema guía al usuario para actuar correctamente.

Especificaciones del sistema

Componentes

  • Sensor de monóxido
  • Microcontrolador
  • Alarma
  • Aplicación móvil

Aplicaciones
