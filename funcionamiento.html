Funcionamiento del sistema

Detección, análisis, alerta y respuesta

Cómo trabaja SentinelGas paso a paso

SentinelGas convierte datos del entorno en información útil para prevenir accidentes, activar una advertencia rápida y asistir al usuario con una respuesta clara.

Proceso del sistema

El funcionamiento puede entenderse como una secuencia continua de detección, análisis y respuesta.

1. Detección

Los sensores analizan constantemente el ambiente en busca de niveles peligrosos.

2. Procesamiento

El microcontrolador interpreta los datos y detecta situaciones de riesgo.

3. Alerta

Se activa una alarma y se envía una notificación al usuario.

4. Acción

El sistema guía al usuario para tomar la mejor decisión posible.

Respuesta del sistema

SentinelGas no solo detecta, sino que acompaña al usuario con información útil para actuar correctamente.

🪟

Monóxido

El objetivo es reducir el riesgo y mejorar la toma de decisiones.
